Info

This is the Info Page! Here, you can see information on how to use this service.

With this project, I aim to create a platform that students can use to maintain organization. The goal is to have a central place where students can manage their schoolwork without having to switch between multiple websites. This platform will have similar functionality to apps like My Study Life, Trello, and myHomework. This means that users will be able to enter upcoming assignments, exams, and other information to keep track of. The app will send reminders to the user when the designated date is close. What will set it apart is that it will allow students to integrate Canvas for use in the application. This means that when they link their Canvas account, it will be able to display schedule, upcoming assignments, current grades, and more. It will have a calendar feature that shows the academic calendar for the current semester, along with any important dates, deadlines, or holidays.

Adding a Task

When you want to add a task on the Home page, enter a name, details, subject, and due date for the task, then click Add Task.

Canvas Integration

The Canvas Integration feature allows you to add tasks directly from your Canvas Todo List!

  1. Type in your Canvas username
  2. Type in your Canvas password
  3. Type in your Canvas authentication token
    • In order to generate a token, navigate to Canvas settings and under Approved Integrations, click "New Access Token"
  4. Click Login
